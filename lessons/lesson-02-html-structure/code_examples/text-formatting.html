Line Breaks and Horizontal Rules

Sometimes you need a line break within a paragraph.
The <br> tag creates a line break without starting a new paragraph.
It's useful for addresses, poems, or anywhere you need precise line control.

Text Emphasis and Importance

Use strong to indicate important text that should be emphasized.

Use em to add emphasis or stress to certain words.

You can even combine them: very important emphasized text.

Other Text Elements

The <code> element is used for inline code snippets.

Use mark to highlight text as if with a highlighter.

The small element represents side comments or fine print.

For subscript: H2O (water) and superscript: E=mc2 (Einstein's equation).

Quotations

For short inline quotes, use the quote element with automatic quotation marks.

For longer quotes that should be displayed as a block, use the blockquote element. This is typically indented and styled differently from regular paragraphs.

Best Practices for Text

  • Use semantic elements: Choose elements based on meaning, not appearance
  • Don't overuse line breaks: Let CSS handle most formatting
  • Structure content logically: Use headings and paragraphs to organize information
  • Keep paragraphs focused: Each paragraph should cover one main idea

Note: While HTML provides basic text formatting, CSS is the preferred method for controlling the visual appearance of text on web pages.
