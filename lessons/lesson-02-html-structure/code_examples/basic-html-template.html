Welcome to HTML Structure

This is a basic HTML template that demonstrates the proper structure of an HTML5 document.

Document Structure Components

Every HTML document should include:

1. DOCTYPE Declaration

The <!DOCTYPE html> declaration tells the browser this is an HTML5 document.

2. HTML Root Element

The <html> element wraps all content and includes the language attribute.

3. Head Section

Contains metadata, title, and links to external resources.

4. Body Section

Contains all visible content that appears on the webpage.

This template provides a solid foundation for any HTML document.
