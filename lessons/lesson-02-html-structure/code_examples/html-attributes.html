Attribute Best Practices

Naming Conventions

  • Use descriptive, meaningful names for IDs and classes
  • Use kebab-case (hyphens) for multi-word names
  • Keep names concise but clear
  • Avoid using spaces in attribute values (use quotes if needed)

Example with Multiple Attributes
