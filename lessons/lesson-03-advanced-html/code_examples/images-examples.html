Image Best Practices

File Organization

Organize images in a logical folder structure:

/images/
  /photos/
    profile.jpg
    team.jpg
  /icons/
    home.svg
    contact.svg
  /logos/
    company-logo.png
    partner-logos/

Naming Conventions

  • Use descriptive, lowercase names
  • Replace spaces with hyphens
  • Include dimensions if needed

Examples:

  • hero-banner-1920x1080.jpg
  • profile-photo-john-smith.jpg
  • social-media-icons.png

Performance Considerations

  • Optimize file sizes for web delivery
  • Use appropriate formats for content type
  • Consider lazy loading for images below the fold
  • Provide multiple sizes for responsive design
