CSS Syntax and Structure

Welcome to CSS! This page demonstrates basic CSS syntax, selectors, and how styles are applied to HTML elements.

CSS Rule Structure

Every CSS rule consists of a selector and a declaration block:

selector {
    property: value;
    property: value;
}

Example Breakdown

Here's how this heading is styled:

h3 {
    color: #34495e;        /* Text color */
    margin-top: 30px;      /* Space above */
    margin-bottom: 15px;   /* Space below */
}

Types of Selectors

1. Element Selectors

Target all elements of a specific type:

p {
    color: black;
    font-size: 16px;
}

2. Class Selectors

Target elements with a specific class attribute (notice the dot prefix):

.intro {
    background-color: #ecf0f1;
    padding: 20px;
}

3. ID Selectors

Target a specific element with an ID (notice the hash prefix):

#main-content {
    max-width: 800px;
    margin: 0 auto;
}

4. Attribute Selectors

Target elements based on their attributes:

input[type="text"] {
    border: 2px solid #bdc3c7;
    padding: 8px;
}

5. Pseudo-class Selectors

Target elements in specific states. Try hovering over this link!

a:hover {
    color: #e74c3c;
    text-decoration: underline;
}

CSS Properties and Values

Properties define what you want to style, values define how:

Common Property Types

Color Properties:

  • color - text color
  • background-color - background color

Typography Properties:

  • font-family - font typeface
  • font-size - size of text
  • font-weight - boldness of text

Layout Properties:

  • margin - space outside element
  • padding - space inside element
  • border - border around element

CSS Comments

Comments help document your CSS code:

/* This is a single-line comment */

/*
   This is a multi-line comment
   that can span several lines
   and explain complex styling decisions
*/

Grouping Selectors

Apply the same styles to multiple selectors by separating them with commas:

h1, h2, h3, h4, h5, h6 {
    font-family: 'Arial', sans-serif;
    margin-bottom: 15px;
}

This is more efficient than writing separate rules for each heading level!

Inline CSS Example

This paragraph uses inline CSS styling:

Inline styles are applied directly to elements using the style attribute. While they have high specificity, they're harder to maintain and should be used sparingly.

Note: Inline styles override external and internal CSS due to higher specificity.

Remember: Good CSS is organized, commented, and follows consistent naming conventions!
