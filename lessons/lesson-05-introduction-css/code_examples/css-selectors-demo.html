Basic Selectors

Element Selectors

This paragraph is styled using an element selector that targets all p elements.

Class Selectors

This paragraph uses the "important" class for special styling.

This div uses the "box" class for border and padding.

ID Selectors

This section has a unique ID and gradient background.

Combinator Selectors

Descendant Selectors

This paragraph is inside a div with class "examples" and gets special styling.

This nested paragraph is also affected by the descendant selector.

.examples p { color: #34495e; font-style: italic; }

Child Selectors

I'm a direct child - I get styled!
I'm a grandchild - I don't get the child selector styling.
.parent > .direct-child { background-color: #d5dbdb; }

Adjacent Sibling Selector

This heading has special styling

This paragraph immediately follows an h3 and gets bold, purple text.

This paragraph doesn't immediately follow an h3, so it's normal.

h3 + p { font-weight: bold; color: #8e44ad; }

General Sibling Selector

This div has class "trigger"

I'm a sibling that gets styled!

I'm also a sibling that gets styled!
.trigger ~ .sibling-target { background-color: #f0f8f0; }

First Letter and First Line

This paragraph demonstrates the ::first-letter pseudo-element, which creates a large drop cap effect for the first letter of the paragraph text.

This paragraph uses the ::first-line pseudo-element to make the first line bold and dark. This effect only applies to the first line as it appears in the layout, which may change based on screen size.
