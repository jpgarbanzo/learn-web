Plain HTML

Structure only - no styling or interactivity

What you see:

Browser default styling - functional but not attractive

  • Basic form structure
  • Default browser styles
  • No custom appearance
  • Form submission reloads page

    <input type="text" placeholder="Enter your name">

2

HTML + CSS

Structure + visual design

What's added:

Custom styling with CSS - modern, attractive appearance

  • Custom colors and gradients
  • Rounded corners and shadows
  • Hover effects and transitions
  • Responsive layout with flexbox
HTML Code 
<form class="styled-form">
    <input type="text" class="styled-input" placeholder="Enter your name">
    <button type="submit" class="styled-button">Submit</button>
</form>

3

HTML + CSS + JavaScript

Structure + design + interactivity

What's added:

JavaScript for dynamic behavior - no page reload!

  • Form submission without reload
  • Input validation
  • Dynamic feedback messages
  • Interactive user experience
HTML Code 
<form class="styled-form js-form" id="myForm">
    <input type="text" class="styled-input" id="nameInput" placeholder="Enter your name">
    <button type="submit" class="styled-button">Submit</button>
</form>
<div class="output-message" id="outputMessage"></div>
JavaScript Code 
// Get references to form elements
const form = document.getElementById('myForm');
const input = document.getElementById('nameInput');
const output = document.getElementById('outputMessage');

// Add event listener for form submission
form.addEventListener('submit', function(event) {
    // Prevent default form submission (page reload)
    event.preventDefault();
    
    // Get the input value
    const name = input.value.trim();
    
    // Validate input
    if (name === '') {
        alert('Please enter your name!');
        return;
    }
    
    // Display personalized message
    alert(`Hello, ${name}! Welcome to interactive web development!`);
    
    // Show success message below form
    output.textContent = `âœ¨ Form submitted successfully for: ${name}`;
    output.classList.add('show');
    
    // Clear the input field
    input.value = '';
    
    // Hide message after 3 seconds
    setTimeout(() => {
        output.classList.remove('show');
    }, 3000);
});

HTML (Structure)

Provides the content and semantic structure of the web page. It's the foundation that search engines and screen readers understand.

CSS (Presentation)

Controls how the content looks - colors, layout, typography, animations. Makes the web beautiful and user-friendly.

JavaScript (Behavior)

Adds interactivity and dynamic behavior. Handles user input, manipulates content, and communicates with servers.
